Interactive 3D World Picture Map

Create an interactive 3D globe where users store pictures by country using Globe.gl API

Features: Click countries to upload/view photos, export data for backup and reimport

Teaches API integration, client-side data persistence, and interactive design beyond basic buttons

  1. Learn external APIs not taught in class
  2. Achieve client-side data persistence with localStorage
  3. Experiment with interactive 3D design
  4. Create practical tool for organizing and sharing photos by location
